# Object-detection-using-YOLO
Real-time object detection using YOLOv8 and OpenCV in Python.

# I have built a real time object detection system with YOLOv8, OpenCV in python. 

# Features
- Real time object detection using an external webcam or the laptop camera.
- YOLOv8 object detection
- FPS counter
- clean and modular code
- Windows Webcam bug fixes included
- Begineer Friendly
